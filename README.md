# ConnecTouchClient
## Requirement
  `python v2.x`  
  `$ pip install nfcpy`  
  `$ pip install requests`
## Run
  ```
  $ python -m SimpleHTTPServer  
  $ cd ConnecTouchClient   
  $ python read-nfc.py
  ```
## Run at RasPi
  `$ sudo python read-nfc.py`
